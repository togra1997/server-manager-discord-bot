# Discordサーバー管理ボット

このリポジトリは、Pythonで実装されたDiscordサーバー管理用のボットです。サーバー管理を効率化するための各種コマンドを提供します。

## 主な機能
- Discordサーバー内でのコマンド実行
- サーバー管理の自動化

## ディレクトリ構成
```
discord_bot/
	bot_main.py         # ボットのエントリーポイント
	.env                # Botトークンなどの環境変数
    run-server.sh       # サーバーの起動コマンドを記載するバッシュファイル
	src/
		run_command.py  # コマンド実行ユーティリティ
```

## 使い方
### サーバーの下準備
1. サーバーを起動するためのバッシュファイルを作成する。ファイルの中身としては、サーバーを実行するコマンドのみを記載する\
例:exec 実行するコマンド ← execが必須です

### botの下準備
1. 必要なPythonパッケージをインストールしてください。 requirements.txtは.devcontainerフォルダにあります。
1. DiscordのBotトークンを`.env`ファイルに設定してください。
	 - 例: `TOKEN=あなたのトークン`
1. `bot_main.py` を実行してボットを起動します。


## 注意事項
- Botをサーバーに追加するには、Discord Developer PortalでBotを作成し、必要な権限を付与してください。(スラッシュコマンドが最低限必要です)
- 詳細な設定や拡張方法は、コード内コメントや各ファイルを参照してください。
